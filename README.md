# BACE Kitchen Manager

A professional kitchen management system for BACE residents, maintaining Sattvic cooking standards (No Onion, No Garlic).

## Features

- **50+ Authentic Recipes** - Complete database of Sattvic recipes in Hindi & English
- **Daily Checklist** - Kitchen readiness verification system
- **Recipe Scaling** - Automatic ingredient scaling for any number of residents
- **Feast Planner** - Menu planning and printing capability
- **Bilingual Support** - Full Hindi/English interface

## Live Demo

Visit: [BACE Kitchen Manager](https://your-username.github.io/bace-kitchen/)

## Local Usage

Simply open `BACE Kitchen.html` in any modern web browser.

## Technology

- HTML5, CSS3 (Tailwind CSS), Vanilla JavaScript
- Chart.js for data visualization
- No backend required - fully static
